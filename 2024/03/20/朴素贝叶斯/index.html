朴素贝叶斯法

本文将着眼于简述朴素贝叶斯法的学习过程以及其中部分数学推导。

首先，先定义训练数据集的模式：

T={(x1,y1),(x2,y2),...,(xN,yN)}T = \{(x_1,y_1),(x_2,y_2),...,(x_N,y_N)\} T={(x1​,y1​),(x2​,y2​),...,(xN​,yN​)}

（注：xix_ixi​为n维向量的集合；yiy_iyi​为类标记，共有K种选择）

学习过程可以简述为以下几步：
（1）朴素贝叶斯法的参数估计：根据训练数据集，我们利用极大似然估计，首先估计出先验概率P(Y=ck)P(Y = c_k)P(Y=ck​)：

P(Y=ck)=∑i=1NI(yi=ck)N,k=1,2,...,KP(Y=c_k) = \frac{\sum\limits_{i=1}^{N}I(y_i = c_k)}{N},k = 1,2,...,K P(Y=ck​)=Ni=1∑N​I(yi​=ck​)​,k=1,2,...,K

然后，我们计算利用联合分布公式，计算出条件概率P(X(j)=ajl∣Y=ck)P(X^{(j)}=a_{jl}|Y = c_k)P(X(j)=ajl​∣Y=ck​):

P(X(j)=ajl∣Y=ck)=P(X(j)=ajl,Y=ck)P(Y=ck)P(X^{(j)}=a_{jl}|Y = c_k) = \frac{P(X^{(j)} = a_{jl},Y = c_k)}{P(Y = c_k)} P(X(j)=ajl​∣Y=ck​)=P(Y=ck​)P(X(j)=ajl​,Y=ck​)​

而由于：

P(Y=ck)=∑i=1NI(yi=ck)N,P(X(j)=ajl,Y=ck)=∑i=1NI(xi(j)=ajl,yi=ck)NP(Y=c_k) = \frac{\sum\limits_{i=1}^{N}I(y_i = c_k)}{N},P(X^{(j)} = a_{jl},Y = c_k) = \frac{\sum\limits_{i=1}^{N}I(x_i^{(j)}=a_{jl},y_i = c_k)}{N} P(Y=ck​)=Ni=1∑N​I(yi​=ck​)​,P(X(j)=ajl​,Y=ck​)=Ni=1∑N​I(xi(j)​=ajl​,yi​=ck​)​

上式可化简为：

P(X(j)=ajl∣Y=ck)=∑i=1NI(xi(j)=ajl,yi=ck)∑i=1NI(yi=ck)P(X^{(j)}=a_{jl}|Y = c_k)=\frac{\sum\limits_{i=1}^{N}I(x_i^{(j)}=a_{jl},y_i = c_k)}{\sum\limits_{i=1}^{N}I(y_i = c_k)} P(X(j)=ajl​∣Y=ck​)=i=1∑N​I(yi​=ck​)i=1∑N​I(xi(j)​=ajl​,yi​=ck​)​

综上所述，先验概率P(Y=ck)P(Y=c_k)P(Y=ck​)以及条件概率P(X(j)=ajl∣Y=ck)P(X^{(j)}=a_{jl}|Y = c_k)P(X(j)=ajl​∣Y=ck​)都可以由训练数据集TTT很简单地统计和计算得到。

（2）结合朴素贝叶斯中的假设，X=(x(1),x(2),...,x(n))TX=(x^{(1)},x^{(2)},...,x^{(n)})^TX=(x(1),x(2),...,x(n))T，XXX的每个属性都是相互独立的。因此有：
